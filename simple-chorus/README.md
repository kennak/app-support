# SimpleChorusEffect

AUv3 Stereo Chorus Effect Plugin for macOS and iOS

---

## Overview

SimpleChorusEffect is a stereo chorus plugin featuring four carefully crafted modes.  
From subtle shimmer to rich ensemble textures, it adds natural width and movement to any sound.

---

## Parameters

### Mode

| Mode | Description |
|------|-------------|
| Chorus I | Classic stereo chorus with smooth, warm modulation |
| Chorus II | Deeper chorus with cross-channel delay for added width |
| HiChorus | High-frequency focused chorus with HPF for a bright, airy texture |
| Ensemble | Lush multi-voice ensemble with wide stereo spread |

### Controls

| Parameter | Range | Description |
|-----------|-------|-------------|
| Rate | 0 – 20 Hz | LFO modulation rate |
| Intensity | 0 – 100 | Modulation depth |
| Mix | 0 – 100 | Dry/Wet mix balance |

---

## Compatibility

- macOS 13 or later
- iOS 16 or later
- AUv3 compatible: Logic Pro, GarageBand

## 概要
